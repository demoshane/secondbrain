## Component Inventory

Components in scope for this phase, with implementation notes:

### People Chips (37-03) ‚Äî new component `PersonAutocomplete`

Clone of `TagAutocomplete` with the following differences:
- Fetches from `GET /persons` (returns `{ people: [{ path, name }] }`)
- Displays `name` field in dropdown (not slug/path)
- Stores `name` (display name string) in the people array ‚Äî NOT path
- Chips render with `variant="secondary"` Badge, same as tag chips
- `√ó` remove button: `opacity-0 group-hover:opacity-100 hover:text-destructive` ‚Äî identical to tag chip pattern
- Location: `flex-wrap gap-1 items-center px-4 py-1 border-b` row in NoteViewer, directly below tag chips row
- `data-testid="people-chips"` on the container div
- Individual chip: `data-testid="person-{name}"` (kebab-cased)
