### Pagination shape (locked decision from CONTEXT.md):

**Flask API:**
```python
limit = min(int(request.args.get("limit", 50)), 200)
offset = int(request.args.get("offset", 0))
total = conn.execute("SELECT COUNT(*) FROM notes").fetchone()[0]
rows = conn.execute("SELECT ... FROM notes ORDER BY created_at DESC LIMIT ? OFFSET ?", (limit, offset)).fetchall()
return jsonify({"notes": notes_list, "total": total, "limit": limit, "offset": offset})
```

**MCP tools:**
```python
# 1-based page param
page = max(1, page)
limit = min(limit, 200)
offset = (page - 1) * limit
total_pages = math.ceil(total / limit) if limit else 1
return {"results": [...], "total": total, "page": page, "total_pages": total_pages}
```
