  <action>
    In engine/api.py, add three endpoints after the existing /brain-health route:

    Import at top (if not present): from engine.brain_health import get_empty_notes; from engine.intelligence import list_actions

    GET /inbox:
    - actions_offset = int(request.args.get('actions_offset', 0)), PAGE_SIZE=20
    - source_note = request.args.get('source_note') or None
    - Unassigned actions: query action_items WHERE done=0 AND assignee_path IS NULL, LEFT JOIN dismissed_inbox_items ON d.path=CAST(a.id AS TEXT) AND d.item_type='action' WHERE d.path IS NULL; add AND a.note_path=:source_note if source_note set; ORDER BY created_at DESC; count total first, then LIMIT 20 OFFSET :offset
    - Unprocessed notes: SQL with LEFT JOIN relationships r ON r.source_path=n.path OR r.target_path=n.path, LEFT JOIN dismissed_inbox_items d ON d.path=n.path AND d.item_type='note'; WHERE type IN ('idea','ideas','coding','strategy','personal','projects') AND (tags IS NULL OR tags='[]' OR tags='null' OR tags='') AND created_at >= datetime('now','-14 days') AND r.source_path IS NULL AND d.path IS NULL; ORDER BY created_at DESC; LIMIT 50
    - Empty notes: SELECT path, title FROM notes WHERE (body IS NULL OR TRIM(body)='') AND path NOT IN (SELECT path FROM dismissed_inbox_items WHERE item_type='note'); LIMIT 20 — call get_empty_notes(conn) and post-filter dismissed, OR write the query inline; inline query with dismiss filter is preferred to avoid list truncation.
    - total_count = len(unassigned_actions) + len(unprocessed_notes) + len(empty_notes) (counts after filtering)
    - Return jsonify({unassigned_actions, unassigned_actions_total, unprocessed_notes, empty_notes, total_count})
    - conn.row_factory = sqlite3.Row; use dict(row) to serialize; wrap in try/finally conn.close()
