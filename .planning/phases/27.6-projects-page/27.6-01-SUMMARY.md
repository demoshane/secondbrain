## What Was Built

Two Flask routes added to `engine/api.py` immediately before the `/actions` block:

- `GET /projects` — lists all notes where `type='projects'`, returning `{path, title, updated_at, open_actions}` ordered by `updated_at DESC`. Open actions count via correlated subquery.
- `GET /projects/<path:note_path>` — fetches full project detail `{path, title, body, updated_at, open_actions}`. Uses `_resolve_note_path()` for path-traversal protection (403). Returns 404 if not in DB.

Six unit tests in `tests/test_projects.py` cover: empty list, populated list, open_actions count, detail fetch, 404 not-found, and 403/404 path-traversal.
