    **6. options.js** (per D-12):
    ```javascript
    const DEFAULT_API_URL = 'http://127.0.0.1:37491';

    async function loadOptions() {
      const { apiUrl, defaultTags } = await chrome.storage.sync.get({
        apiUrl: DEFAULT_API_URL,
        defaultTags: '',
      });
      document.getElementById('apiUrl').value = apiUrl;
      document.getElementById('defaultTags').value = defaultTags;
    }

    async function saveOptions() {
      await chrome.storage.sync.set({
        apiUrl: document.getElementById('apiUrl').value.trim() || DEFAULT_API_URL,
        defaultTags: document.getElementById('defaultTags').value.trim(),
      });
      const status = document.getElementById('status');
      status.textContent = 'Saved!';
      setTimeout(() => { status.textContent = ''; }, 1500);
    }

    document.addEventListener('DOMContentLoaded', loadOptions);
    document.getElementById('saveBtn').addEventListener('click', saveOptions);
    ```

    **background.js** (per D-02 — icon click + context menu only):

    1. `chrome.runtime.onInstalled` listener — create 3 context menu items:
       - id: `"capture-page"`, title: `"Save Page to Brain"`, contexts: `["page"]`
       - id: `"capture-selection"`, title: `"Save Selection to Brain"`, contexts: `["selection"]`
       - id: `"capture-link"`, title: `"Save Link to Brain"`, contexts: `["link"]`

    2. `chrome.contextMenus.onClicked` listener:
       - Store `pendingCapture` object in `chrome.storage.session`:
         ```javascript
         { menuItemId, selectionText: info.selectionText, linkUrl: info.linkUrl,
           pageUrl: tab.url, pageTitle: tab.title, timestamp: Date.now() }
         ```
       - Call `chrome.action.openPopup()` as the LAST statement (user gesture chain per Research pitfall 1)

    3. Helper function `getApiUrl()`:
       ```javascript
       async function getApiUrl() {
         const { apiUrl } = await chrome.storage.sync.get({ apiUrl: 'http://127.0.0.1:37491' });
         return apiUrl;
       }
       ```

    **content.js** (non-Gmail section — Gmail handled in Plan 03):

    1. `chrome.runtime.onMessage` listener with two actions:
       - `"extract-article"`: Clone document, run `new Readability(docClone).parse()`, sanitize with `DOMPurify.sanitize(article.textContent)`, return `{ title, textContent, excerpt, url: location.href }`. Must `return true` to keep message channel open.
       - `"extract-selection"`: Return `{ selectionText: window.getSelection().toString(), url: location.href, pageTitle: document.title }`. Must `return true`.

    **popup.js** (per D-04, D-05 — always popup before save):

    1. On `DOMContentLoaded`:
       a. Check `chrome.storage.session` for `pendingCapture`:
          - If found: remove it, populate form based on `menuItemId`:
            - `"capture-page"`: send `"extract-article"` message to active tab content script, fill title/body/url. Pre-suggest type `"note"` (per D-04).
            - `"capture-selection"`: use `selectionText` as body, `pageTitle` as title. Pre-suggest type `"note"`.
            - `"capture-link"`: use `linkUrl` as body content (`URL: {linkUrl}`), derive title from linkUrl. Pre-suggest type `"link"`.
          - If not found (icon click): query active tab, send `"extract-article"` to content script, fill form. Pre-suggest type `"note"`.

       b. Check API connectivity — fetch `GET {apiUrl}/ping` with 2s timeout:
          - Success: show green status bar "Connected"
          - Failure: show red status bar "sb-api unreachable" (per D-10), disable Save button

       c. Load default tags from `chrome.storage.sync` and pre-fill tags field if set

    2. Form submit handler (`save-btn` click):
       a. Gather form values: title, type, body, tags (split by comma, trim)
       b. Determine endpoint:
          - If type is set by user, use `POST /notes` with `{title, type, body, source_url: pageUrl}`
          - Alternative: use `POST /smart-capture` with `{content: body, source_url: pageUrl, source_type: captureType}`
          - Use `/notes` endpoint (simpler, user has already chosen type in dropdown)
       c. POST to `{apiUrl}/notes` with JSON body
       d. On success (201): show "Saved!" message, add to capture history, close popup after 1s
       e. On failure: show error message inline

    3. Cancel button: `window.close()`

    4. Capture history (per D-11):
       - `addToHistory({title, type, url, savedAt})` — store in localStorage, max 10 entries
       - On popup open: render history list if any entries exist, show history section
       - Each history item: title + type badge + relative time
