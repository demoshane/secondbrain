# Phase 41.2 Plan 04: Add Link Note to PeoplePage Connections Section Summary

**One-liner:** Inline Link Note form with search + select in PeoplePage's Related Notes section, POSTing to POST /relationships with rel_type=manual and refreshing backlinks on success.

## What Was Built

Added a "Link Note" capability to the Related Notes CollapsibleSection in PeoplePage:

- **State:** `showLinkNote`, `linkNoteSearch`, `allNotes`, `selectedNoteToLink`, `linkingNote`
- **`handleShowLinkNote`:** Fetches `GET /notes?limit=100` to populate the note selector, resets search and selection state, shows the form
- **`handleLinkNote`:** POSTs to `POST /relationships` with `source_path=selectedPath`, `target_path=selectedNoteToLink`, `rel_type='manual'`; on success refreshes backlinks from `GET /notes/<path>/meta`; shows toast feedback
- **UI:** Toggle between "Link Note" outline button and an inline search-input + select dropdown + Link/Cancel buttons — same UX pattern as "Link Meeting" in ProjectsPage

Imports updated: added `Link` from lucide-react.

## Deviations from Plan
