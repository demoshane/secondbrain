**useNoteActions.ts:**

Change `saveNote` signature to accept title and body:
```ts
const saveNote = useCallback(async (path: string, body: string, title?: string) => {
  const encoded = encodePath(path)
  const payload = title !== undefined
    ? { title_and_body_title: title, title_and_body_body: body }
    : { body }
  const res = await fetch(`${getAPI()}/notes/${encoded}`, {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload),
  })
  return res.ok
}, [])
```

When `title` is provided → uses `title_and_body_*` combined branch (preserves frontmatter, updates both).
When `title` is not provided → uses `{ body }` branch (preserves frontmatter, updates body only).
