# Phase 27.7 Plan 01: Right Panel Playwright Coverage Summary

Two new Playwright test functions added to cover the RightPanel component; gui_brain fixture extended with two seed notes supporting those tests.

## What Was Built

**conftest.py — two new session-scoped seeds in `gui_brain`:**

1. `Test Group` (`type='people'`) — seeds a plural-type note to guard against type confusion (alice-smith regression class where `type='people'` was mishandled).
2. `Test Mention Note` (`type='idea'`, body: `"This note mentions Test Person in the body."`) — provides a note whose body-mention of "Test Person" will trigger the `/notes/<path>/meta` people detection logic.

**test_gui.py — two new test functions:**

1. `test_right_panel_renders` — verifies `data-testid="right-panel"` is visible after opening any note. Exercises the RightPanel container which previously had zero Playwright coverage.
2. `test_right_panel_people_mention` — opens Test Mention Note, waits for `data-testid="people-badge"` with text "Test Person" to appear in the right panel. Direct regression guard for the body-mention detection logic in `/notes/<path>/meta`.
