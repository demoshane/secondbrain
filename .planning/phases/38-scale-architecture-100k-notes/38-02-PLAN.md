  <action>
    Create `engine/sharding.py` with:

    1. `SHARD_MAP` dict mapping note types to subdirectories:
       ```python
       SHARD_MAP = {
           "meeting": "meetings",
           "person": "people",
           "project": "projects",
           "strategy": "strategy",
           "idea": "ideas",
           "personal": "personal",
           "coding": "coding",
           "file": "files",
           "link": "links",
       }
       DEFAULT_SHARD = "notes"
       ```

    2. `get_shard_path(brain_root: Path, note_type: str, filename: str) -> Path`:
       Returns `brain_root / SHARD_MAP.get(note_type, DEFAULT_SHARD) / filename`.
       Creates subdirectory if not exists.

    3. `shard_note(conn, old_path: str, new_path: str) -> None`:
       In a single DB transaction:
       - Verify old file exists on disk (raise FileNotFoundError if not)
       - Create target directory if needed
       - Move file: `Path(new_path).parent.mkdir(parents=True, exist_ok=True)`, then `Path(old_path).rename(new_path)`
       - UPDATE notes SET path=new_path WHERE path=old_path
       - UPDATE note_embeddings SET note_path=new_path WHERE note_path=old_path
       - UPDATE note_tags SET note_path=new_path WHERE note_path=old_path
       - UPDATE note_people SET note_path=new_path WHERE note_path=old_path
       - UPDATE action_items SET note_path=new_path WHERE note_path=old_path
       - UPDATE relationships SET source_path=new_path WHERE source_path=old_path
       - UPDATE relationships SET target_path=new_path WHERE target_path=old_path
       - UPDATE audit_log SET note_path=new_path WHERE note_path=old_path
       - conn.commit()
       Note: If any UPDATE fails, the transaction rolls back and the file move is NOT reverted
       (caller must handle). Use `try/except` with `conn.rollback()` + move file back on failure.

    4. `shard_all_notes(conn, brain_root: Path, dry_run: bool = True) -> list[dict]`:
       For each note in DB, compute target shard path. If different from current path,
       add to moves list. If `dry_run=False`, execute `shard_note()` for each.
       Returns list of `{"old_path": ..., "new_path": ..., "moved": bool}`.
