  <action>
    Create `engine/backup.py`:

    1. Constants:
       ```python
       from pathlib import Path
       KEY_PATH = Path.home() / ".config" / "second-brain" / "backup.key"
       HNSW_FILENAME = "brain.hnsw"
       LABEL_MAP_FILENAME = "label_map.json"
       ```

    2. `_get_or_create_key() -> bytes`:
       - Create parent dir if needed. If KEY_PATH exists, read and return.
       - Otherwise generate via `Fernet.generate_key()`, write, chmod 0o600, return.

    3. `backup_brain(brain_root: Path, db_path: Path, backup_dir: Path) -> Path`:
       - Create backup_dir if needed.
       - Build tar.gz in memory (io.BytesIO) containing:
         - All .md files under brain_root (arcname relative to brain_root, prefix "notes/")
         - db_path as "brain.db"
         - brain_root / ".meta" / HNSW_FILENAME as "brain.hnsw" (if exists)
         - brain_root / ".meta" / LABEL_MAP_FILENAME as "label_map.json" (if exists)
       - Encrypt tar bytes with Fernet(_get_or_create_key())
       - Write to backup_dir / f"brain-{timestamp}.enc" (timestamp: %Y%m%dT%H%M%SZ)
       - Return path to .enc file

    4. `restore_brain(enc_path: Path, target_root: Path, target_db: Path) -> dict`:
       - Decrypt enc_path with Fernet(_get_or_create_key())
       - Extract tar.gz:
         - "notes/" prefix -> target_root
         - "brain.db" -> target_db
         - "brain.hnsw" -> target_root / ".meta" / HNSW_FILENAME (if in archive)
         - "label_map.json" -> target_root / ".meta" / LABEL_MAP_FILENAME (if in archive)
       - Return {"notes_restored": count, "db_restored": True, "hnsw_restored": bool}

    5. `check_backup_health(backup_dir: Path, warn_days: int = 7) -> dict`:
       - Find newest .enc file in backup_dir
       - Return {"last_backup": ISO timestamp or None, "stale": bool, "age_days": int or None}

    6. `backup_main()`: CLI entry point. Calls backup_brain with default paths. Prints result.

    7. `restore_main()`: CLI entry point. Two-step confirm_token pattern:
       - Parse `--enc-file` argument (required)
       - First run: print warning "This will overwrite your brain. Confirm token: {token}"
       - Second run with `--confirm {token}`: execute restore_brain
       - Token: random 8-char hex, stored in module-level dict with 60s expiry

    Create fake .md files, a fake brain.db, test encrypt/decrypt round-trip.
