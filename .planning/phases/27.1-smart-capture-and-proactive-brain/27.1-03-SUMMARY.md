## What Was Built

**check_capture_dedup(title, body, conn, threshold=0.92)** in `engine/capture.py`:
- Embeds `title\nbody` via `embed_texts()`, queries `note_embeddings` with `vec_distance_cosine`
- Returns list of `{"path": str, "similarity": float}` dicts, most similar first, up to 5
- Best-effort: returns `[]` on any exception (model not loaded, extension absent, empty table)
- Never raises, never blocks capture

**sb_capture() in `engine/mcp_server.py`** updated with:
- New `confirm_token: str = ""` parameter
- Dedup check block after `get_connection()` but before `capture_note()`
- Returns `{"status": "duplicate_warning", "similar": [...], "confirm_token": tok}` when dedup fires
- `_consume_token()` call when token provided; raises `TOKEN_EXPIRED` for invalid/expired tokens
- Clean fallthrough: if no match or embeddings unavailable, capture proceeds normally
