## Accomplishments

- `find_uv()`: shutil.which with ~/.local/bin fallback, raises FileNotFoundError
- `install_global_cli()`: uv tool install --editable --force; conditionally calls update-shell when ~/.local/bin not in PATH
- `write_plist()`: plistlib.dump with Label, ProgramArguments, KeepAlive=True, stdout/stderr log paths; accepts plist_path override for test hermetics
- `load_launchd_agent()`: launchctl bootout (absorbs error) then bootstrap gui/<uid> idempotently; raises RuntimeError on bootstrap failure
- `install_hook()`: /usr/bin/git rev-parse validation + core.hooksPath config; raises ValueError for non-git paths
- `main()`: argparse --cli/--launchd/--hooks; default no-args runs all three
