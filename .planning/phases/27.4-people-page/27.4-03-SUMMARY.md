## What Was Built

**UIContext.tsx:** View union extended from `'notes' | 'actions'` to `'notes' | 'actions' | 'people'`.

**TabBar.tsx:** People tab added as third entry in TABS array; renders as `tab-people` data-testid.

**App.tsx:** Ternary `notes ? layout : <ActionsPage />` replaced with `notes ? layout : actions ? <ActionsPage /> : <PeoplePage />` if-else chain. PeoplePage import added.

**PeoplePage.tsx (138 lines):**
- Left column (w-72): filter input + shadcn Table with Name / Actions / Last Note columns; active row highlighted with `bg-accent`
- Right column (flex-1): "Select a person to view details" when nothing selected; person detail when selected
- Four collapsible Section components: Note body (react-markdown + remark-gfm), Meetings, Backlinks, Open Actions
- Parallel fetch via `Promise.all([/notes/<enc>, /notes/<enc>/meta, /actions?assignee=<enc>])`
- Meetings extracted from backlinks where path contains `/meetings/`; remaining backlinks shown separately
- "Open in Notes" calls `openNote(selectedPath)` then `setCurrentView('notes')`
- All required data-testid attributes: `people-page`, `note-body-section`, `meetings-section`, `backlinks-section`, `actions-section`
