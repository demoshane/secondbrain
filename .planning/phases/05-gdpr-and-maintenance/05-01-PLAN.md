  <behavior>
    - test_forget_deletes_person_file: person file at brain_root/people/{slug}.md does not exist after call
    - test_forget_deletes_sole_reference_meeting: meeting file with people=[slug] does not exist after call
    - test_forget_spares_shared_meeting: meeting file with people=[slug, other] still exists after call
    - test_forget_cleans_backlinks: a note body containing the slug string has that line removed after call
    - test_search_zero_after_forget: search_notes(slug, conn) returns [] after call
    - test_fts5_rebuild_after_forget: conn.execute was called with SQL containing "rebuild" after call
  </behavior>
  <implementation>
Implement forget_person(slug, brain_root, conn) following the deletion order from RESEARCH.md:

1. Parse meeting files in brain_root/meetings/ using python-frontmatter. Classify each as sole-reference (remaining people list empty after removing slug) or shared.
2. Walk all .md files in brain_root (excluding person file and sole-reference meetings). For files containing slug in body text, remove lines that contain the slug. Write back using pathlib (FOUND-12).
3. Delete sole-reference meeting files from disk (Path.unlink).
4. Delete person file from disk (Path.unlink, if exists).
5. DELETE FROM notes WHERE path = exact person path OR path in exact list of sole-reference meeting paths. Use exact paths, NOT LIKE patterns (avoids Pitfall 5 from RESEARCH.md — slug too broad).
6. DELETE FROM relationships WHERE source_path = person path OR target_path = person path OR source_path IN (sole-ref meeting paths) OR target_path IN (sole-ref meeting paths).
7. DELETE FROM audit_log WHERE note_path = person path OR note_path IN (sole-ref meeting paths).
8. INSERT INTO notes_fts(notes_fts) VALUES('rebuild') — GDPR-02.
9. conn.commit().
10. INSERT into audit_log: event_type='forget', note_path=NULL, detail='person:{slug}', created_at=utcnow ISO8601.
11. conn.commit().
12. Return dict: {deleted_files: list[str], cleaned_backlinks: list[str], errors: list[str]}.

Implement main() using argparse: positional arg "person" (slug or "First Last"), converts spaces to hyphens, lowercases, calls get_connection() + init_schema() + forget_person() + prints summary.

Error message rule: never include file content in error strings — only type(e).__name__ (GDPR-05).
  </implementation>
</feature>
