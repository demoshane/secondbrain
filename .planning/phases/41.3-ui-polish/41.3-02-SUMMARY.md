## What Was Built

Rewrote `popup.css` from the original light theme (`#fff` background, `#1a1a2e` text) to match the Second Brain app's dark Visily design system. Applied the same hex token values used by the main app's CSS variables:

- **Body background:** `#0f1117` (matches `--background`)
- **Input/select/textarea backgrounds:** `#191f2d` (matches `--input` / `--muted`)
- **Primary button:** `#5b6cf7` hover `#4a5be0` (matches `--primary`)
- **Foreground text:** `#d4dae8` (matches `--foreground`)
- **Borders/dividers:** `#2a3245` (matches `--border`)
- **Muted labels:** `#7a8499` (matches `--muted-foreground`)
- **Secondary button:** `#1e2535` bg / `#2a3245` border (matches `--secondary`)
- **Focus ring:** `rgba(91, 108, 247, 0.35)` (matches `--ring` with alpha)
- **Error states:** `#2d1b1b` bg / `#f87171` text / `#dc2626` border
- **Success states:** `#1b2d1b` bg / `#4ade80` text / `#16a34a` border
- **History type badge:** `#1e2a4a` bg / `#818cf8` text (indigo tint matching primary family)
- **Select chevron:** SVG data-URI arrow, `appearance: none` for consistent dark styling
- **popup.html:** Added `🧠` identity marker before "Save to Brain" h2 text
