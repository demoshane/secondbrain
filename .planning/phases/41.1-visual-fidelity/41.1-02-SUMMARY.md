## What Was Built

### Task 1: ActionsPage Source-Note Grouping

- `groupBySource(items)` groups `ActionItem[]` by `note_path` into `Map<string, ActionItem[]>`
- `extractTitle(notePath)` converts path like `meetings/2026-03-28-standup.md` → `2026 03 28 standup`
- `allCollapsed` state + "Collapse All" / "Expand All" toggle button with `ChevronDown`/`ChevronRight` icons
- Per-group collapse state (`Map<string, boolean>`) — groups use `allCollapsed` as fallback; toggling `allCollapsed` resets individual overrides
- Each group renders as a bordered card with a clickable header (title + item count badge) and a collapsible body of `ActionItemRow` components
- "Sort by: Source Note" label placed above the grouped content area

### Task 2: MeetingsPage Date Chips

- `formatDateChip(dateStr)` parses ISO date string, returns `{ month: "MAR", day: "28" }` or null on failure
- Month abbreviation uses `MONTH_ABBR` lookup array + `.toUpperCase()` (though already uppercase)
- Chip styled as `inline-flex flex-col items-center justify-center w-10 h-10 rounded-md bg-primary/15 text-primary shrink-0`
- Chip renders left of title in a `flex items-center gap-2.5` row
- Removed old `{m.meeting_date && (<div className="text-xs text-muted-foreground mt-0.5">...` plain-text date line
