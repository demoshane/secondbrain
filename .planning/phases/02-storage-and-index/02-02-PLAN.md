SQL without type filter:
```sql
SELECT n.path, n.type, n.title, n.created_at, bm25(notes_fts) AS score
FROM notes_fts
JOIN notes n ON notes_fts.rowid = n.id
WHERE notes_fts MATCH ?
ORDER BY bm25(notes_fts)
LIMIT ?
```

SQL with type filter:
```sql
SELECT n.path, n.type, n.title, n.created_at, bm25(notes_fts) AS score
FROM notes_fts
JOIN notes n ON notes_fts.rowid = n.id
WHERE notes_fts MATCH ?
  AND n.type = ?
ORDER BY bm25(notes_fts)
LIMIT ?
```

After fetching rows, insert audit log entry:
```python
conn.execute(
    "INSERT INTO audit_log (event_type, note_path, detail, created_at) VALUES (?, ?, ?, ?)",
    ("search", None, query, datetime.datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ")),
)
conn.commit()
```

Return list of dicts. If rows is empty, return [].

CRITICAL: ORDER BY bm25(notes_fts) — ASC is the default and gives best-first. Do NOT add DESC.
CRITICAL: audit INSERT uses detail=query column, not note_path — this avoids logging a full note path for a search event.
