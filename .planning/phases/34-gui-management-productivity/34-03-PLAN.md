@app.post("/people")
def create_person():
    data = request.get_json(force=True)
    name = data.get("name", "").strip()
    if not name:
        return jsonify({"error": "name is required"}), 400
    role = data.get("role", "").strip()

4. `GET /people/<path>/links` — Cascade link counts for delete warning:
```python
@app.get("/people/<path:note_path>/links")
def get_person_links(note_path):
    abs_path, _ = _resolve_note_path(note_path)
    with get_connection() as conn:
        meeting_count = conn.execute(
            "SELECT COUNT(*) FROM note_people WHERE person_path = ?",
            (str(abs_path),)
        ).fetchone()[0]
        action_count = conn.execute(
            "SELECT COUNT(*) FROM action_items WHERE assignee_path = ?",
            (str(abs_path),)
        ).fetchone()[0]
    return jsonify({"meeting_count": meeting_count, "action_count": action_count})
```

5. `DELETE /people/<path>` — Delete person with cascade:
```python
@app.delete("/people/<path:note_path>")
def delete_person(note_path):
    abs_path, _ = _resolve_note_path(note_path)
    with get_connection() as conn:
        # Clear assignee references (per Pitfall 7 from RESEARCH)
        conn.execute("UPDATE action_items SET assignee_path = NULL WHERE assignee_path = ?", (str(abs_path),))
        # Remove from note_people junction
        conn.execute("DELETE FROM note_people WHERE person_path = ?", (str(abs_path),))
        conn.commit()
    # Delegate to existing note delete logic (handles FTS, embeddings, audit)
    from engine.forget import forget_note
    forget_note(str(abs_path))
    _broadcast({"type": "notes_changed"})
    return jsonify({}), 200
```

6. Similarly add `DELETE /meetings/<path>` and `DELETE /projects/<path>` — simpler cascade (just remove note via forget_note + broadcast). Meetings/projects don't have assignee references.

**Path encoding note:** The frontend uses `encodeURIComponent(entityPath)` which encodes slashes as `%2F`. Flask's `<path:note_path>` route receives the URL-decoded value (literal slashes).
