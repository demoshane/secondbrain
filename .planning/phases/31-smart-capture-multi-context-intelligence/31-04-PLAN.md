       - After save loop completes, process `links` field on each note dict:
         ```python
         # Post-save: resolve links and create relationships
         path_map = {s["index"]: s["path"] for s in succeeded}
         for i, note in enumerate(notes or []):
             if i not in path_map:
                 continue
             links = note.get("links", [])
             for slug in links:
                 # Resolve slug to path: try title match first
                 row = conn.execute(
                     "SELECT path FROM notes WHERE LOWER(title) = LOWER(?) LIMIT 1",
                     (slug.replace("-", " "),)
                 ).fetchone()
                 if not row:
                     # Try path LIKE match
                     row = conn.execute(
                         "SELECT path FROM notes WHERE path LIKE ? LIMIT 1",
                         (f"%{slug}%",)
                 ).fetchone()
                 if row:
                     now = datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%SZ")
                     # Bidirectional link
                     conn.execute("INSERT OR IGNORE INTO relationships VALUES (?,?,?,?)",
                                  (path_map[i], row[0], "link", now))
                     conn.execute("INSERT OR IGNORE INTO relationships VALUES (?,?,?,?)",
                                  (row[0], path_map[i], "link", now))
         conn.commit()
         ```
