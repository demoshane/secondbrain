**Decimal Phases:**
- Phase 41.1: Visual Fidelity (inserted after Phase 41 for Visily gap closure)

**Key Decisions:**
- Relative paths in DB (Phase 32) — eliminates /var vs /private/var macOS symlink issues
- SQLite triggers for junction table sync (Phase 48.1) — eliminates dual-write fragility
- Groq for fast AI queries, Ollama for privacy (Phase 44) — user-configurable routing
- Multi-pass decomposer architecture (Phase 43) — modular, independently testable passes
- hnswlib ANN index (Phase 38) — fast nearest-neighbor search at scale, sqlite-vec fallback

**Issues Resolved:**
- Phase 39 audit: 31 findings across 6 categories, all remediated by Phases 40–48.1
- Phase 47: 4 long-standing test failures fixed
- Phase 41.2: Critical note save data loss bug fixed (frontmatter overwrite)
